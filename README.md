# SMART FHIR Clinical Dashboard

A healthcare interoperability application built using React, TypeScript, Node.js, PostgreSQL, and FHIR APIs.

## Overview

This project demonstrates how SMART on FHIR applications can securely access and visualize clinical data through standardized healthcare APIs. The dashboard provides clinicians with a unified view of patient information, including encounters, medications, observations, and clinical notes.

## Features

* SMART on FHIR authentication workflow
* OAuth 2.0 secure authorization
* Patient record visualization
* Clinical observations dashboard
* Medication history viewer
* Encounter timeline
* Healthcare data analytics
* Responsive React user interface
